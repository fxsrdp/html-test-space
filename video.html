How We Got Here

Method

Nur ein <video>-Tag in einem <div> mit border-radius:50% und overflow:hidden. Das Video selbst via object-fit:cover füllt den Kreis. Kein JS, kein Canvas-Drawing, keine Transkodierung.

CSS

.video-circle { border-radius:50%; overflow:hidden }
.video-circle video { object-fit:cover; width:100%; height:100% }

Das Video behält sein natives 1:1-Aspect-Ratio, wird via CSS auf 72/120/300px skaliert. object-fit:cover sorgt für randlose Füllung des Kreises.

HTML Attributes

autoplay — startet sofort (braucht muted in Chrome)
loop — 30s Loop, nahtlos
muted — Stummschaltung für Autoplay-Policy
playsinline — verhindert iOS-Vollbild

Browser Support

Safari ✅ · Chrome ✅ · Edge ✅ — HEVC nativ
Firefox ❌ — braucht VP9-Fallback für Produktion
iOS ✅ — playsinline verhindert Vollbild

Performance Notes

2.3 MB bei 30s — ~78 KB/s Bitrate. Hardware-dekodiert auf Apple Silicon, kein CPU-Overhead bei 72×72.
Für Produktion: auf 10s Loop kürzen (800 KB) oder VP9-Transcode (~1.2 MB für Firefox).
