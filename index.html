technique

Depth-to-Image Pipeline

Blender 5.1→Cycles Depth→EXR→16-bit PNG→Pad Square→ComfyUI /upload/image→ControlNet Depth→SDXL KSampler→Editorial Output

Product α-Mask

Render subject meshes in isolation — EEVEE, film_transparent, 1024², HORIZONTAL sensor_fit. Export as RGBA PNG with embedded alpha.

Full-Scene Depth

Cycles 128 samples, Z‑Pass → Normalize → Invert → EXR → 16‑bit PNG. Same camera settings as α‑mask for pixel-perfect overlay.

SDXL Generation

ControlNet Depth SDXL (strength 0.85) + KSampler (dpmpp_2m karras, CFG 7.0, 20 steps) → 1024² editorial scenes. MPS ~3 min each.

Offline α‑Slice

Python/Pillow: product mask × SDXL output → product_*.png. Inverted mask × SDXL → env_*.png. Beauty render sliced for base pass. All RGBA with embedded alpha.

HTML Stacking

z‑order: Environment(z1) → Analysis(z2, blend modes) → Product(z3). Radio toggles for env/product. Multi‑toggle for analysis. Zero CSS masks.
